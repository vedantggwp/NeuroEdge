Motion Guidelines

Property Value Use Case
--duration-fast 150ms Hover states, focus rings, tooltips
--duration-normal 300ms Card lifts, button transitions, theme switch
--duration-slow 500ms Page reveals, section entrance, gauge fill
--ease-out cubic-bezier(0.16, 1, 0.3, 1) Most transitions — fast start, smooth settle
--ease-spring cubic-bezier(0.34, 1.56, 0.64, 1) Interactive elements — slight overshoot for energy
Stagger delay 80-100ms per child Sequential content reveals (cards, stats, list items)

Motion Principles

Principle
Purposeful, not decorative
Every animation communicates something — entrance, emphasis, feedback, or state change. If it doesn't serve a purpose, remove it.
Principle
Fast feels responsive
Hover and focus transitions at 150ms. Page-level animations at 300-500ms. Never exceed 800ms for any single animation.
Principle
Respect prefers-reduced-motion
Wrap all animations in a @media query. Users who set prefers-reduced-motion: reduce should see no motion beyond opacity fades.
